# Reality-IRC (Reality-Messenger Core)

**Reality-IRC** — это легковесная, модульная реализация IRC-сервера на языке Python, полностью соответствующая стандарту **RFC 1459**. Проект разработан с упором на минимализм, высокую скорость работы и простоту расширения через систему плагинов.

Ядро оптимизировано для запуска в ограниченных средах, таких как **Android (Termux)**, и не требует сторонних зависимостей.

---

## 🔥 Ключевые особенности

### 🏗 Модульная архитектура команд
В отличие от монолитных серверов, здесь каждая IRC-команда (`JOIN`, `PRIVMSG`, `KILL` и т.д.) является отдельным модулем в директории `protocol/`.
- Динамическая загрузка модулей через `importlib`.
- Возможность добавлять или обновлять функционал без остановки основного ядра.

### 🛡 Надежная работа с сетью
- **TCP Stream Handling:** Реализован алгоритм склеивания TCP-пакетов через буфер. Это гарантирует, что сообщения не будут потеряны или обрезаны, даже если они приходят по частям.
- **Неблокирующий I/O:** Использование системного вызова `select.select` позволяет серверу эффективно обрабатывать десятки одновременных подключений в одном потоке.

### 🔒 Безопасность
- **Path Traversal Protection:** Загрузчик команд фильтрует ввод и разрешает выполнение только тех модулей, имена которых состоят из букв латинского алфавита.
- **Валидация прав:** Реализована система IRC-операторов (`MODE +o`), позволяющая разграничивать доступ к критическим командам управления сервером.

---

## 🛠 Реализованные команды

Сервер поддерживает полный цикл взаимодействия пользователя с сетью:

- **Авторизация:** `NICK`, `USER`, `QUIT`, `CAP` (совместимость с WeeChat/HexChat).
- **Каналы:** `JOIN`, `PART`, `LIST`, `NAMES`, `TOPIC`, `KICK`.
- **Общение:** `PRIVMSG`, `NOTICE` (личные сообщения и чаты каналов).
- **Администрирование:** `OPER`, `KILL`, `MODE`.
- **Информационные:** `WHO`, `WHOIS`, `MOTD`, `INFO`, `VERSION`, `ADMIN`, `LUSERS`, `LINKS`.
- **Служебные:** `PING`, `PONG` (поддержание активности сессии).
